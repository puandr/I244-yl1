Important things to create a HTML page

  1. In HTML 5 you have to define document type by starting it with <!DOCTYPE html>
  2. HTML document have to have a head, where you can define title and charset
  3. And of course the bodyitself, otherwise why to learn HTML? :)
  4. Don't forget to close tag that have to be closed, its a very common bug!

Basic HTML text format

HTML offers enough tools to format text:
  • Simple text
  • Strong text or bold text
  • Emphasized text or italic text
  • Marked text
  • Small text
  • Deleted text
  • Inserted text or underlined text
  • Subscript text and Superscript text
